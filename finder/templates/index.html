How you can use Represent

From running advocacy campaigns to making legislative information more accessible, organizations use Represent to match postal codes and addresses to electoral districts and elected official contact information. To learn more about how specific civil society organizations use Represent, check out our blog. There are many ways you can benefit from Represent's resources. Here are some suggestions:

On your website

You can set up your own instance of Represent using our API. For example, Citizen Bridge uses the API to display riding maps on their website and the "ridings to postal code" data to sign-up users. Instructions are available in the documentation and Open North is here to help!

In a campaign

Advocacy groups can access accurate information about Canadian representatives based on the postal code or address of their supporters and unlike other postal code lookup services, Represent is completely free and open source. For example, we helped Environment Defense directly integrate Represent into their online forms so that when a supporter joins their letter-writing campaign, it is automatically sent to the correct representative. To facilitate this type of integration, Open North and its volunteers have created two Drupal Modules, for Webforms and CiviCRM users.

Develop a new app

The Represent database and API can be used to create lots of new apps like the Windows Phone app, MyRepresentatives, which tracks the activity of MPs in Parliament. If you are interested in using Represent and want to set up a private instance of the service to ensure your project has reliable access to the server, contact Open North to discuss Service Level Agreements.
